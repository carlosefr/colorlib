COLORLIB README
-------------------------
colorLib is a library for managing and manipulating colors in Processing.
It was created in 2008 by Andreas Kšberle and Jan Vantomme.
For more info on the people behind this project visit these websites.
http://www.eskimoblood.de/
http://www.pixelplayground.eu/

LICENSE
-------------------------
colorLib is open source software and is licensed under the MIT license.

INSTALLATION
-------------------------
Drag the colorLib folder to the libraries folder of your Processing installation.

USAGE
-------------------------
Go to Sketch > Import Library > colorlib.
See the documentation for more info on how to use colorlib.
